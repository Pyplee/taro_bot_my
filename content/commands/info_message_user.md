*Это справка по формату сообщений в телеграм от пользователя:*

*Жирный текст*
```md
**Жирный текст**
```

_Курсивный текст_
```md
__Курсивный текст__
```

__Подчеркнутый текст__
```md
--Подчеркнутый текст--
```

~Зачеркнутый текст~
```md
~~Зачеркнутый текст~~
```

||Спойлер||
```md
||Спойлер||
```

*Создание кодового оформления:*
```js
const hi = () => return 'hi';
```
Используются такие кавычки \`\`\` сверху и снизу кода\.
